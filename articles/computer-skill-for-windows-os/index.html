    Windows 系统下的小技巧

    • 修电脑
    最后更新： 2018-01-08
    阅读次数： 次
    许可协议： 知识共享署名-非商业性使用-禁止演绎 4.0 国际许可协议

    电脑故障

    1. 开机后，桌面有明显的卡顿现象。

      检查现在系统默认使用的显卡是不是自己电脑的独立显卡。（因为有时候系统更新后，你自己对显卡的配置可能会被重置）

    2. 电脑开机后，风扇不再像以前那样安静稳定，而是转个不停，声音还挺大，并且就算啥也不做，电脑也会有明显的高于室温的温度。

      要是遇到这种情况，可以考虑换个电脑风扇了。淘宝上搜索自己电脑型号的风扇即可。当然，在换风扇的同时，要是再为电脑的 CPU、显卡重新换一次散热硅胶更佳。

    将自己喜欢的程序添加到鼠标右键菜单

    这里以将 sublime 添加到右键菜单为例！

    1. 打开注册表（运行 regedit）
    2. 定位到 HKEY_CLASSES_ROOT*\shell
    3. 在 shell 下新建一个项,我命名为 sublime
    4. 再在 sublime 下新建一个项,命名为 command
    5. 然后修改右边的数据（这里要将 sublime 的路径添加上去,并且在路径最后面要添加一个 %1）
      • eg: E:\Installed_Software\Sublime Text 2\sublime_text.exe %1

    注意，要是程序换了路径后，添加到右键菜单的程序就不起作用了，请删掉 shell 下与程序相关的项，然后按照上面的步骤重新设置。

    为 Windows 系统的 cmd 添加命令的别名

    有目的地设置合适的别名，可以大大提高在命令行下的工作效率。

    1. 新建一个文件，我这里命名为 alias.bat，放在 C 盘根目录下

    2. 在刚刚新建的文件中写入自定义的命令别名，然后保存文件

      • 为一条长命令设置别名，比如返回上一层目录。
      • 为多条命令设置别名，比如生成 hexo 静态页面并启动本地服务器。

      注意：与 Linux 不同，windows 下为执行多条命令设置别名时，要用 ^& 分开，而不是 &&。

    3. 打开注册表，定位到 HKEY_LOCAL_MACHINE\Software\Microsoft\Command Processor，鼠标右键新建字符串值，名称设为 AutoRun，值为第二步中的文件的路径 C:\alias.bat，保存退出。

    4. 重新打开 cmd，即可看到上面的别名自动被设置了。

    要是不想在打开 cmd 的时候看到别名的自动设置，可以更改 alias.bat 文件的 doskey 为 @doskey。这下，每次启动 cmd 就不会看到别名的自动设置了（不影响别名的正常使用）。
